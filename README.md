🧠 ¿Qué es NovaDL?

NovaDL es una herramienta de escritorio diseñada para usuarios que quieren control total sobre sus descargas.

✔ Rápida
✔ Minimalista
✔ Potente
✔ Sin basura innecesaria

🚀 Características principales
[+] Multi-URL Downloader
[+] Interfaz rápida y limpia
[+] Compilación a .exe
[+] Portable ready (.zip)
[+] Auto Releases con GitHub Actions
[+] Arquitectura modular

🧬 Instalación (modo dev)
git clone https://github.com/tu-usuario/NovaDL.git
cd NovaDL
pip install -r requisitos.txt
python main.py
